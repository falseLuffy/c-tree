### 基于vue开发的树形图

### 可选参数
```
   data:Array 
   property:{name:'name',children:'children'}
   icon:无
```
### 使用方法：
```
   <c-tree :data="data"></tree>

   import CTree from './c-tree.js'
   {
    components:{
       CTree
    }
   }
```
   或
```
   <script src="./c-tree.js"></script>
   <script>

        template:'<div>'+

        data:function(){

                      {
                         name: '开始开始开',
```
